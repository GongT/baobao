##### LinuxErrorCode

Linux POSIX 标准错误代码枚举，值为字符串形式的错误码名称 (如 `'ENOENT'`)。

包含 POSIX 标准错误码及 Linux 扩展错误码，可与 Node.js `ErrnoException.code` 配合使用。

**类型:** `enum LinuxErrorCode`

常见值: `EPERM`、`ENOENT`、`EACCES`、`EEXIST`、`EISDIR`、`ENOTDIR`、`ECONNREFUSED`、`ETIMEDOUT` 等。
